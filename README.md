# Todo_list 介绍

Vue-cil+Vue3+Ts+localStorage 非常基础的 todolist
![image](https://user-images.githubusercontent.com/52476806/133882920-430d172a-385d-4802-af48-530cb8d66261.png)

## Demo

https://hehaoxuan.github.io/TodoList-Vue3-Ts/

## 基本功能

- 增加任务
- 删除任务
- 一键勾选所有
- 清除已完成任务
- 页面本地存储

## 项目启动

```
npm install
```

## 启动服务:

```
npm run serve
```
